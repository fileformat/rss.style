What is this?

Every website that wants to be found should have a sitemap.xml file that Google (and everyone else) can use to help find and index all the pages.

By default, these files look horrible when a human looks at them in a browser. But there is a simple way to make them look better: Sitemap Style!

How do I use it?

Add a line to the top of your sitemap.xml file with one of the styles here on Sitemap Style:

<?xml-stylesheet type="text/xsl" href="https://www.sitemap.style/xslt/vanilla-water.xslt" ?>

Available styles

  • vanilla.xslt - just a plain bullet list of pages
  • Vanilla + classless: still a plain list, but with a classless CSS files
    • vanilla-picnic.xslt - with picnic.css
    • vanilla-pico.xslt - with pico.css
    • vanilla-pure.xslt - with pure.css
    • vanilla-water.xslt - with water.css
